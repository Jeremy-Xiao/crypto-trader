## 策略评估指标

| 指标 | 良好标准 | 警戒线 | 淘汰标准 |
|------|----------|--------|----------|
| 年化收益率 | > 30% | 15-30% | < 15% |
| 最大回撤 | < 15% | 15-25% | > 25% |
| 夏普比率 | > 1.5 | 0.5-1.5 | < 0.5 |
| 胜率 | > 55% | 45-55% | < 45% |
| 盈亏比 | > 1.5 | 1.0-1.5 | < 1.0 |

## 迭代流程

```
策略迭代循环：
  1. 监控运行状态
  2. 定期绩效评估
  3. 识别退化信号
  4. 决策：继续/暂停/淘汰
  5. 如淘汰：启用备选策略
  6. 研发新备选策略
```
