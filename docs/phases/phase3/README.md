## 风控规则配置

```yaml
risk_control:
  # 日亏损限制
  daily_loss_limit_pct: 0.03  # 3%
  
  # 单币种持仓限制
  single_position_limit_pct: 0.20  # 20%
  
  # 连续止损暂停
  consecutive_loss_limit: 5  # 连续5次
  
  # 单笔最大亏损
  single_trade_loss_limit_pct: 0.01  # 1%
  
  # 自动平仓触发
  auto_close_on_limit: true
```
