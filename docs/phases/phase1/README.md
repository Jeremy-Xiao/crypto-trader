## 策略示例伪代码

```
策略名称：双均线趋势跟随

开仓条件：
  当 1小时线收盘价 > EMA50 且 RSI > 50
  用账户权益的 2% 开多

平仓条件：
  当 跌破 EMA50 或 单笔亏损达 1% 止损
  
止盈：
  当 单笔盈利达 3% 止盈
```
