# 阶段1：策略原型与回测

**时间周期**：第3-5周

**目标**：定义清晰的交易逻辑，通过历史数据回测验证有效性

---

## 任务清单

### 1. 确定策略类型
- [ ] 选择策略模型（建议从经典模型起步）
  - 双均线交叉
  - 布林带回归
  - 网格交易
  - 动量突破

### 2. 用伪代码定义规则
- [ ] 明确开仓条件
- [ ] 明确平仓条件
- [ ] 仓位计算公式
- [ ] 止损止盈规则

### 3. 回测框架搭建
- [ ] 选择回测引擎（backtrader/vectorbt）
- [ ] 模拟滑点、手续费
- [ ] 设置手续费参数（现货0.1%，合约taker 0.05%）

### 4. 历史数据清洗
- [ ] 获取历史数据（至少一个牛熊周期）
- [ ] 处理数据缺口
- [ ] 过滤极端插针

### 5. 回测与优化
- [ ] 计算绩效指标：
  - 年化收益率
  - 最大回撤
  - 夏普比率
  - 胜率
  - 盈亏比
- [ ] 样本外数据验证（最近30%时间）
- [ ] 参数不超过2-3个

### 6. 敏感性与压力测试
- [ ] 参数变化测试
- [ ] 增加交易成本测试
- [ ] 极端行情模拟（"5.19"/"3.12"）

## 策略示例伪代码

```
策略名称：双均线趋势跟随

开仓条件：
  当 1小时线收盘价 > EMA50 且 RSI > 50
  用账户权益的 2% 开多

平仓条件：
  当 跌破 EMA50 或 单笔亏损达 1% 止损
  
止盈：
  当 单笔盈利达 3% 止盈
```
